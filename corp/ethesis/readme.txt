How to harvest E-thesis PDFs?

1) Go to e-thesis website https://helda.helsinki.fi/handle/10138/17738
2) Search for the type of theses you want to harvest, e.g. Lisensiaatinty√∂t / Biotieteet: https://helda.helsinki.fi/handle/10138/18075/browse?type=dateissued&sort_by=3&order=DESC&rpp=20&offset=15
3) Modify the url: Set results per page to 100 (rpp=100) and make new url by increasing the offset value by 100, e.g.

https://helda.helsinki.fi/handle/10138/18070/browse?order=DESC&rpp=100&sort_by=3&etal=-1&offset=0&type=dateissued
https://helda.helsinki.fi/handle/10138/18070/browse?order=DESC&rpp=100&sort_by=3&etal=-1&offset=100&type=dateissued
https://helda.helsinki.fi/handle/10138/18070/browse?order=DESC&rpp=100&sort_by=3&etal=-1&offset=200&type=dateissued

If there are 1240 theses, the offset should go up to this number.

4) Copy this list of urls into "urlit.txt" (see the example file attached).
5) Run the getPDF Python script (Python 2.8).
6) DO NOT lose the logfile.txt generated by the harvesting script as it contains all the metadata
7) OCR the PDF files
8) Run ethesis2vrt.py [pdfname.pdf] for each pdf file (Python 3+). "logfile.txt" must be in the same folder.
9) Now you should have the VRT with proper metadata.

Plese see the makecorpus.sh and the naming conventions before importing stuff into Korp! E.g. Master's theses from the faculty of pharmacy should be named ethesis_ma_far (remember not to overwrite the existing corpora!)
